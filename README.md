# Madar-Style Landing Replica

Internal frontend reference recreation of a premium logistics landing page.

## Run Locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Build

```bash
npm run build
```

## What Is Included

- Next.js + TypeScript implementation.
- Reusable React components for header, hero, chapter sections, partners, stats, case study, demo form, footer, cookie banner, and responsive rotation notice.
- Safe placeholder SVG/CSS visuals instead of proprietary images, logos, fonts, or brand assets.
- Request a Demo form with frontend validation, disabled/error/success states, custom selects, and local-only submit handling.
- Desktop, tablet, and mobile responsive behavior.
- Audit and QA docs:
  - `AUDIT.md`
  - `VISUAL_QA.md`

## Notes
